EagleRide

EagleRide is a project developed in 24 hours for HackTheHeights 2024, designed to provide a streamlined and efficient ride-sharing solution tailored for campus communities. I completed the project with teammates Chase Hansen, Trevor Lavallee, and Ryan McLoughlin. Built with simplicity and accessibility in mind, EagleRide connects users with available rides in real time, making transportation easier and more convenient. Github Link

Tools Used: Python, JavaScript, HTML/CSS, Flask, SQL, Google Maps API, Figma
